How region extraction works:
1. Capture full page → perspective-correct using detected sheet edges
2. Crop predefined rectangular regions (calibrated to InBody 570 layout)
3. OCR each region independently with Vision framework
4. Parse each region with section-specific extractors (knows field order + expected format)
5. Score confidence per field using Vision's per-observation confidence values
