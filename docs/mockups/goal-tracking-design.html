04 · Data model
@Model
final class Goal {
  var id: UUID
  var metric: String // TrendMetric rawValue (e.g. "weight", "bodyFatPct")
  var targetValue: Double
  var targetDate: Date?
  var startValue: Double
  var startDate: Date
  var status: String // active | completed | abandoned
  var completedDate: Date?
  var createdAt: Date
}

One active goal at a time. Completed/abandoned goals persist as history. startValue captures where you were when the goal was set. Goal line only renders on the chart when viewing the goal's metric.
