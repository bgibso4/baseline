Design Decisions

1. Date chip first, warning second -- user verifies scan identity before reviewing field accuracy.

2. No tab bar -- scan flow is a fullScreenCover (iOS convention for pushed modal flows).

3. Editable value cells -- every value sits in a subtle card background (--card) signaling tappability. Tap opens decimal pad. Missing fields show dashed border + unit label so user knows what to type.

4. Three field states: Normal (card bg, white text), Low confidence (amber border + amber text), Missing (dashed border, gray "--", unit still visible).

5. Field order mirrors the paper -- sections appear in the same order as InBody 570 printout.

6. Segmental table -- mass + sufficiency % in compact two-column layout. Each cell independently editable.

7. Retry banner -- inline "Retry Scan" button. Save disabled (elevated bg, tertiary text) until required fields present.
