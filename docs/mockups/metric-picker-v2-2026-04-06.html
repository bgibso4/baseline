03 · Interaction rules (reference)

Compare OFF: Tap any row → becomes primary, sheet dismisses, chart updates.

Compare ON: Primary is locked (accent check, non-interactive). Tap an unset row → becomes secondary (amber check). Tap current secondary → nothing. Tap a different row → swaps secondary.

Changing primary: Turn compare OFF, pick new primary, turn compare back ON.

Chip updates: Single metric → icon + name. Dual metric → stacked icons + "Primary · Secondary" dual name.

Metrics shown: Only metrics with at least 1 recorded data point. Grouped: Core, Body Composition, Segmental Lean, Segmental Fat, Measurements.

Compare-only sections: Previous Period + Program only visible when compare is ON. "Last year" greyed if <12 months of data. "Apex phases" disabled with "Soon" badge.

Animation pass (flagged): Chip chevron rotation, chip pulse on sheet open, sheet spring animation, metric row selection ripple — deferred to animation pass.

Hero dimming: Weight metric: accent when today's entry exists, textTertiary when not. Non-weight scan metrics: always colored (scans aren't daily).
