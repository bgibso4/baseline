# Baseline — site branch

This is the `gh-pages` branch of the Baseline iOS app repo. It hosts the public privacy policy and terms of service on GitHub Pages.

Live URLs:

- https://bgibso4.github.io/baseline/
- https://bgibso4.github.io/baseline/privacy/
- https://bgibso4.github.io/baseline/terms/

App source lives on the `main` branch.

## Editing

Edit `privacy.md` or `terms.md` and push to this branch. GitHub Pages will rebuild within a minute or two.

Bump the "Last updated" date at the top of each page when making substantive changes, and call out the change in the app's release notes.
